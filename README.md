# 📖 Sobre o Projeto

A Asuka é uma assistente virtual desenvolvida como um projeto de estudo e aperfeiçoamento em Python, com foco em arquitetura modular, processamento de linguagem natural, integração com APIs e automação de tarefas.

O objetivo do projeto é criar uma assistente inteligente capaz de compreender comandos em linguagem natural, interagir com diferentes serviços e oferecer uma experiência cada vez mais próxima de um assistente pessoal.

---

# ✨ Funcionalidades

Atualmente a Asuka é capaz de:

- 🎤 Reconhecer comandos em linguagem natural
- 🧠 Interpretar intenções do usuário
- 📂 Abrir programas instalados no computador
- 🌐 Integrar diferentes APIs
- 🔍 Consultar informações sobre Pokémon através da PokéAPI
- 📚 Utilizar um banco de dados local para consultas rápidas
- ⚙️ Trabalhar com uma arquitetura modular para facilitar futuras expansões
